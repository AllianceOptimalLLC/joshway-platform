JOSHWAY Lovable Codebase Security Review

Generated July 1, 2026 · Read-only archive analysis · 4 apps · 25 findings

The JOSHWAY Lovable monorepo contains four independent Vite/React apps, each bound to its own Supabase project, with no shared auth layer. Academy, Connect Hub, and Base Camp use Supabase Auth with role-based RLS (relatively sound). Mission Control uses three parallel auth models backed by broadly permissive RLS on core tables. Cross-app integration is limited to Connect Hub's broken Academy invite card.
